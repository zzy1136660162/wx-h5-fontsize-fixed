# FontsizeFixed 按钮

### 介绍

FontsizeFixed 是一个在微信h5中固定字体大小的vue组件

### 引入

```js
import Vue from 'vue';
import { FontsizeFixed } from 'wx-h5-fontsize-fixed';

Vue.use(FontsizeFixed);
```

## 代码演示

### 基础用法

```html
<fontsize-fixed>{{slot}}</fontsize-fixed>
```

## API


### Slots

| 名称    | 说明     |
| ------- | -------- |
| default | 默认插槽 |
